# ReadMe

Erstellung eines R-Packets zur Implementierung Zufälliger Wälder.\
\
Random Forests sind Verfahren, die zur Klassifikation und Regression benutzt werden können.\
Ein Random Forest besteht aus mehreren Entscheidungsbäumen.\
Aufgabenstellung:\
• Implementieren Sie das Gierige Verfahren, vgl. Definition 6.15, 6.16\
• Implementieren Sie Cost-Complexity Pruning, vgl. Definition 6.18\
• Implementieren Sie den Bagging-Algorithmus für Regressions- und Klassifikationsbäume, vgl. Definition 6.25, 6.26, 6.28\
• Implementieren Sie die Random-Forest-Algorithmen für Regression und Klassifikation, vgl. Definition 6.52.\
• Erstellen Sie eine Shiny-App zur einfacheren Bedienung.\
• Illustrieren Sie Ihre Verfahren anhand von Beispielen\
