# ReadMe

Creation of an R Package for Implementing Random Forests .\

Random forests are methods that can be used for classification and regression.\
A random forest consists of multiple decision trees.\
Task description:\
• Implement the greedy procedure, cf. Definitions 6.15, 6.16\
• Implement cost-complexity pruning, cf. Definition 6.18\
• Implement the bagging algorithm for regression and classification trees, cf. Definitions 6.25, 6.26, 6.28\
• Implement the random forest algorithms for regression and classification, cf. Definition 6.52.\
• Create a Shiny app for easier use.\
• Illustrate your methods with examples\
